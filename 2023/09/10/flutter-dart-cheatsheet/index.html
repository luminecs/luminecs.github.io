Command line

  • flutter --version: Check the version of Flutter and Dart.

  • flutter doctor: Check Flutter development environment is okay.

  • flutter upgrade: Upgrade Flutter and Dart SDK.

  • flutter create ./my_app --project-name my_app --empty: Create a new Flutter project.

  • flutter pub add http: Add http package to current project from pub.dev.

  • flutter pub get: Get all dependencies of the current project from pub.dev.

  • flutter pub run build_runner build: Run build_runner to generate some dart code.

  • flutter devices: List connected devices that are available.

  • flutter emulators: List connected emulators that are available.

  • flutter emulators --launch {emulator name}: Launch a emulator.

  • flutter run -d {device name}: Run app on selected device.

  • q: Close app.

  • flutter test: Run the Flutter test.
